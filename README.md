⚙️ Getting Started
1. Clone the Repository
```
git clone https://github.com/<your-username>/mohdhusaindarji-portfolio.git
cd mohdhusaindarji-portfolio
```

2. Install Dependencies
```
npm install
```

3. Start Development Server
```
npm run dev
(or)
npm start
```

4. Build for Production
```
npm run build
```
